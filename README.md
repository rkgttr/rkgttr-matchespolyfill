### How to Install

```sh
$ npm install rkgttr-matchespolyfill --save-dev
```
or

```sh
$ yarn add rkgttr-matchespolyfill --dev
```

### Getting Started

```js
import 'rkgttr-matchespolyfill';
```

or

```js
require('rkgttr-matchespolyfill');
```
